SP GlowPetals — Image Assets Guide
====================================

Place your image files in this folder.

REQUIRED FILES:
---------------
logo.png          → Website logo (recommended: 180x60px, transparent bg)
ti.png            → Browser tab icon / favicon (recommended: 64x64px)

BANNER IMAGES (Hero Slider):
------------------------------
banner1.png       → Hero slide 1 (recommended: 1920x900px)
banner2.png       → Hero slide 2
banner3.png       → Hero slide 3

(Add more via Admin Panel → Banners → Add Banner)

CATEGORY IMAGES:
-----------------
cat-1.png through cat-7.png  → Category thumbnails (600x800px portrait)

OFFER SECTION:
--------------
offer1.png        → Left offer banner background (1000x500px)
offer2.png        → Right offer banner background (1000x500px)

ABOUT PAGE:
-----------
about1.png        → About section photo (800x600px)

PRODUCT IMAGES:
---------------
Place product images in a subfolder:
  assets/images/products/product-name.png

In Admin → Add Product → enter path:
  assets/images/products/rose-candle.png

RECOMMENDED SIZES:
------------------
Product images   → 800x800px (square)
Banner images    → 1920x900px
Category images  → 600x800px portrait
Logo             → 400x120px transparent PNG
Favicon          → 64x64px PNG
